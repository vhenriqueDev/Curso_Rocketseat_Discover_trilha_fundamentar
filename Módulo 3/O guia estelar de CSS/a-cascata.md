# A Cascata (cascading)

A escolha do browaser de qual regra aplicar, caso haja muitas regras para o mesmo elemento.

- Seu estilo é lido de cima para baixo.

É levado em consideração 3 fatores

1. Origem do estilo
2. Especificidade
3. Importância
