# Selectors

Conecta um elemento HTMl com o CSS

## Tipos

- Global selector `*`
- Element/Type selector `h1, h2, p, div`
- ID Selector `#box, #container`
- Class Selector `.red, .m-r`
- Attribute selector, Pseudo-class, Pseudo-element, e outros

```css
* {
  margin: 0;
}

#container {
  width: 200px;
}

.m-40 {
  margin: 40px;
}

h1 {
  color: green;
  font-size: 60px;
  background: gray;
}
```
