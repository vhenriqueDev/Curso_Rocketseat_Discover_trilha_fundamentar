# cores

Usamos CSS para alterar cores do nosso documento.

## tipos

- background-color (para caixas)
- color (para textos)
- border-color (para-caixas)
- outros...

## Valores

Podemos definir os valores por

- palavra-chave (blue, transparent)
- hexadecimal (#990011)
- funções: rgb, rgba, hsl, hsla
